# Eve-Chatbot-Using-Flask
An chatbot uses Flask REST API and communicates in Vietnamese via audio.


#REQUIREMENTS:
Flask==2.2.3

tensorflow==2.15.0

pickle5==0.0.11

pip install -r requirements.txt
